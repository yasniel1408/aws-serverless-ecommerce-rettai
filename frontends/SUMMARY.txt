═══════════════════════════════════════════════════════════════════
🚀 DESARROLLO LOCAL
═══════════════════════════════════════════════════════════════════

Opción 1: Script automático
─────────────────────────────
cd frontends
./dev.sh

Esto:
1. Instala dependencias si no existen
2. Inicia ambas apps en paralelo
3. Main:  http://localhost:3000
4. Admin: http://localhost:3001

Opción 2: Manual (2 terminales)
─────────────────────────────
Terminal 1:
cd frontends/web-rettai
npm install
npm run dev

Terminal 2:
cd frontends/web-rettai-admin
npm install
npm run dev

⚙️ CONFIGURACIONES IMPORTANTES
═══════════════════════════════════════════════════════════════════

web-rettai/next.config.js:
──────────────────────────
{
  output: 'export',         // Static export
  images: { unoptimized },  // Para S3/Amplify
  trailingSlash: true       // URLs con /
}

web-rettai-admin/next.config.js:
────────────────────────────────
{
  output: 'export',         // Static export
  basePath: '/admin',       // Todas las rutas bajo /admin
  images: { unoptimized },
  trailingSlash: true
}

📋 PRÓXIMOS PASOS

4. Deploy automático con Amplify

5. Acceder a:
   https://rettai.com      → Main site
   https://rettai.com/admin → Admin panel
